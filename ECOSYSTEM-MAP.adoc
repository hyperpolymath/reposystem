:toc: preamble
:toclevels: 2

[.lead]
Working map of the reposystem-centered ecosystem. This separates confirmed relationships from user-asserted links pending evidence.

== Core Set (initial)

* reposystem (source-of-truth dependency graph) — 100% complete
* git-hud (formerly gitvisor) — forge HUD — 40% complete
* git-seo — discovery artifacts — 100% complete (v0.4.0)
* git-dispatcher — automation/dispatch core — 20% complete
* git-batcher — bulk repository operations (replaces git-dropper) — planned
* gitbot-fleet — bot fleet coordinator — 35% complete
* scaffoldia — scaffolding/templating — extracted to forge repo + gitlinked into reposystem (2026-06-26)
* .git-private-farm — private repo management — status unknown
* stateful-artefacts-for-gitforges — forge artefact hydration — extracted to forge repo + gitlinked into reposystem (2026-06-26)

**For detailed status, completion percentages, and integration points, see link:ECOSYSTEM-STATUS.adoc[Ecosystem Status].**
