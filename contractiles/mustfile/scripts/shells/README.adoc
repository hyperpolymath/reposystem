== Usage

Each script attempts to locate the `must` binary in the following order:

1. System PATH (`must`)
2. Local build (`./bin/must`)

If neither is found, it exits with an error.

== Making Scripts Executable

[source,bash]
----
chmod +x scripts/shells/*
----

== Customization

These scripts are minimal by design. For project-specific customization,
copy the appropriate script and modify as needed.
