== Relationship to Reposystem

Reposystem integrates with ISA via:

- **PanLL Panel-W**: Vexometer scores feed into the ecosystem health dashboard
- **Aspect annotations**: ISA metrics can be attached as weighted aspects on repos/edges
- **Slot system**: ISA is a provider for the `efficacy-measurement` slot
