== Scope and Standard

Evaluates the `reposystem` monorepo against *Component Readiness
Grades v2.0 (STRICT)*, 2026-03-30 revision. reposystem is
self-declared *C* with a `TEST-NEEDS.md` scorecard headed
"CRG Grade: C ‚Äî ACHIEVED 2026-04-04" and a one-line `crg-grade = "C"`
in STATE.a2ml. The audit checks the claim against on-disk evidence,
estate dogfooding signal, and the v2.0 "honest D > dishonest B" clause.

reposystem is explicitly named in the workspace MEMORY.md Monorepo
Directory Map, so the audit treats it as a monorepo: the top-level
crate (`Cargo.toml` ‚Üí `reposystem`) is graded together with the 20+
sibling subproject directories (`bitfuckit/`, `claim-forge/`,
`avatar-fabrication-facility/`, `git-morph/`, `git-seo/`, `forge-ops/`,
`gui/`, `tui/`, `scaffoldia/`, `stateful-artefacts/`, `total-recall/`,
`total-upgrade/`, `recon-silly-ation/`, `rpa-elysium/`, `vexometer/`,
